# Changelog

## 0.3.0

- Added direct Apple identity-token login for platform bridges.
- Added Apple desktop OAuth sign-in through the Hive Axyl auth server.
- Extended the desktop loopback callback server to receive Apple form POST callbacks without changing Google or Facebook query callbacks.

## 0.2.0

- Added Facebook desktop OAuth sign-in through the Hive Axyl server.
- Shared the loopback callback server between Google and Facebook desktop sign-in.
- Excluded player session tokens from desktop OAuth start and completion requests.

## 0.1.0

- Initial public UPM package release.
